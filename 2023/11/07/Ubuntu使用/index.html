Ubuntu使用

  • 0、常用命令

更新软件源

sudo apt update

升级软件包

sudo apt upgrade

下载软件包

sudo apt install package_name #推荐使用

或使用

sudo apt-get install package_name -y #-y 可选，将自动确认安装过程中的询问

移除/卸载软件包

sudo apt remove --purge package_name # remove将保留配置文件；purge将同时删除配置文件

解压.tar文件

tar -xvf package_name.tar

解压.tar.gz文件

tar -zxvf package_name.tar.gz

删除文件

rm file_name

删除目录

rm -rf file_path

授予终端管理员权限

sudo su

移动/重命名文件

mv /old_path/old_file_name /new_path/new_file_name
  • 1、安装.deb包
dpkg -i ./package_name.deb
  • 2、miniconda的安装与卸载

安装

下载最新版miniconda

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

安装miniconda

bash ./Miniconda3-latest-Linux-x86_64.sh

安装后执行

Source .bashrc

注：询问yes|no时，全选yes

卸载

先找到miniconda

find -name miniconda3

再将其删除

rm -rf miniconda3

打开 ～/.bashrc文件

vim ~/.bashrc

将其中和miniconda相关的语句删除

  • 3、安装最新稳定版node.js

使用apt安装，然后使用n工具升级

sudo apt update

sudo apt install nodejs

# 自行安装npm工具
sudo apt install npm

# 升级npm
sudo npm install npm -g

使用nNode版本管理工具安装最新版node.js

sudo npm install n -g

# 下载最新稳定版
sudo n stable

# 查看已下载版本
sudo n ls

  • 5、安装WPS-Office

先安装WPS-Office本身

sudo apt install wps-office

然后下载字体

sudo apt install ttf-mscorefonts-installer # 安装必需字体

注：安装过程需要同意协议，通过tab和enter键来选中和确认

再按需下载可选字体

sudo apt install fonts-wqy-zenhei fonts-wqy-microhei
  • 6、安装MySQL
    安装（8.0版本）
    sudo apt install mysql-server

验证已安装

sudo systemctl status mysql

以root身份登录mysql

mysql -u root -p

由于没有设置密码，直接回车即可

在mysql环境下，键入以下命令，来设置root用户密码

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '你的密码';

然后刷新权限(mysql环境下)

FLUSH PRIVILEGES;

  • 8、安装Redis

更新软件源

sudo apt update

安装redis

sudo apt install redis-server

检查服务的状态

sudo systemctl status redis-server

提升为sudo权限

sudo su

编辑配置文件

vim etc/redis.conf

找到bind 127.0.0.1 ::1，去掉前面的注释符号#；
如果不存在则添加;
修改后保存文件并退出。

重启redis服务

sudo systemctl restart redis-server

验证Redis服务器是否正在监听端口6379

ss -an | grep 6379

>>>得到类似如下信息

tcp  LISTEN 0   511   0.0.0.0:6379   0.0.0.0:*
tcp  LISTEN 0   511      [::]:6379      [::]:*
